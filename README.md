## Assignments Overview

### Assignment 1: Rule-Based Agent
- [cite_start]**Objective**: Understand agent architecture using simple rule-based logic[cite: 25, 26].
- [cite_start]**Implementation**: Uses keyword matching to identify intents like "calculate" or "date"[cite: 29, 32, 33].

### Assignment 2: Tool-Using Agent
- [cite_start]**Objective**: Enable the agent to call external functions/APIs[cite: 45, 46].
- [cite_start]**Tools Included**: Calculator, Weather (mocked), and Text Summarizer[cite: 50, 51, 52].

### Assignment 3: LLM-Based Agent
- [cite_start]**Objective**: Integrate a language model to replace hard-coded logic for tool selection[cite: 66, 67, 69].
- [cite_start]**Features**: Prompt-based decision making and execution logging[cite: 84, 78].

### Assignment 4: Multi-Step Agent (Planning)
- [cite_start]**Objective**: Design an agent that breaks complex problems into smaller, sequential steps[cite: 88, 89, 91].
- [cite_start]**Example Task**: Finding an average of numbers and summarizing the result in one request[cite: 94].
