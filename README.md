# Agentic AI Lab - 2026

This repository contains a series of assignments focused on building autonomous AI systems that use reasoning, planning, and tool usage to perform tasks.

## Project Structure
The repository is organized into daily progress folders as per the lab requirements:

- **day1/**: Rule-Based AI Agent 
- **day2/**: Tool-Using Agent with modular architecture 
- **day3/**: LLM-Based Agent with automated decision making
- **day4/**: Multi-Step Agent capable of task decomposition and planning

---

## Assignments Overview

### Assignment 1: Rule-Based Agent
- **Objective**: Understand agent architecture using simple rule-based logic.
- **Implementation**: Uses keyword matching to identify intents like "calculate" or "date".

### Assignment 2: Tool-Using Agent
- **Objective**: Enable the agent to call external functions/APIs.
- **Tools Included**: Calculator, Weather (mocked), and Text Summarizer.

### Assignment 3: LLM-Based Agent
- **Objective**: Integrate a language model to replace hard-coded logic for tool selection.
- **Features**: Prompt-based decision making and execution logging.

### Assignment 4: Multi-Step Agent (Planning)
- **Objective**: Design an agent that breaks complex problems into smaller, sequential steps.
- **Example Task**: Finding an average of numbers and summarizing the result in one request.

---

## Technical Requirements
- **Language**: Python 3.9+
- **Key Concepts**: Task decomposition, function calling, and modular programming.
